# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this tool:

1. DO NOT open a public issue
2. Email me: [your email] or DM me on Reddit
3. I will respond within 48 hours

## Data Safety

This tool:
- Runs entirely in your browser
- Does NOT send any data to servers
- Does NOT store any data locally
- Does NOT use analytics or tracking
- Is completely open source

## Verification

You can test the tool's safety yourself:

1. Open your browser's DevTools (F12)
2. Go to the Network tab
3. Use the tool - you will see NO network requests

## Dependencies

- jsPDF (loaded from CDN)
- jspdf-autotable (loaded from CDN)

Both are reputable open-source libraries.
